# AgriSmart-Device
This is a simple device developed to improve acuuracy of model for soil nutirent prediction and help farmers get accurate data.

#What is there is this repo
Code for Esp-32 which self hosts the sensor data and updates every 2 seconds 

#Sensors used 
>ESP-32
>Soil Moisture Sensor
>Temparature and Humidity Sensor
